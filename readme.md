<br/>

clone project
```terminal
git clone https://github.com/kajikasama/sambel-project.git
cd sambel-project/
npm install 
```
open 2 terminal
<br>
running a backend server 
```terminal
cd backend/
nodemon index.js
```

running frontend client
```terminal
cd frontend/
npm start
```

enjoy ~
